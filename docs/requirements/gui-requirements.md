1\. Экран и технические ограничения

* Разрешение: 240x320 пикселей (Ориентация: Landscape / Горизонтальная — 320 по ширине, 240 по высоте).  
* Стиль графики: Flat Design (абсолютно плоский дизайн), Pixel Perfect (строгая привязка к пиксельной сетке для удобства переноса в код C++).  
* Табу: Никаких градиентов, размытий, прозрачностей (Alpha-blending) и сложных теней.  
* Разделение данных: Без использования сетки линий (таблиц). Разделение достигается размером шрифтов, отступами (Negative Space) и цветом.

2\. Цветовая палитра и иерархия

* Фон: Глубокий черный (\#000B16).  
* Фон оверлеев/всплывающих блоков: Темно-серый (\#121212 или \#1A1A1A) для создания объема без использования границ.  
* Цвет режимов (Акцентное кольцо \- обводка/фон):  
  * \#AEB9D0/\#1A202D (Серый) — Режим ожидания (IDLE).  
  * \#00BCD1/\#00212D (Неоновый синий) — Автоматический режим (ТСТАТ).  
  * \#A3552E/\#241812 (Оранжевый) — Ручной режим (РУ).  
  * \#FFB000/\#191C14 (Янтарный/Желтый) — Процесс продувки и остывания (STOP).  
* Шрифты: Крупные рубленые шрифты стиль Aldrich. В качестве иконок используются символы из шрифта FontAwesome 7+ Pro duotone.

3\. Компоновка интерфейса (Layout)

Центр экрана (The Core):

* Элемент: Тонкое кольцо диаметром 200 пикселей. Цвет кольца меняется в зависимости от активного режима работы.  
* Внутри кольца:  
  * *Верхняя строка (мелко:34):* Текущая температура в помещении (Белый цвет).  
  * *Средняя строка (крупно:72):* Температура теплообменника (Оранжевый цвет).  
  * *Нижняя строка (мелко:34):* Влажность в процентах (Светло-серый цвет).

Угловые блоки (Corner Widgets):

*Блоки прижаты к углам экрана. Их внешние границы образуют прямые углы, чтобы дизайн гармонировал с прямоугольным экраном 320x240. Каждый блок имеет подпись.*

* Верхний левый: Индикатор режима.  
* Нижний левый: Индикатор расхода топлива насосом.  
* Верхний правый: Скорость вентилятора в виде заполняющихся фоном по мере увеличения скорости прямоугольников \+ цифра выбранной ступени: 1, 2, 3 или 4\.  
* Нижний правый: Секундомер, в случае включенной свечи показывает “секунды с момента включения”/”максимальное время работы свечи из конфига”.

Быстрое меню (Overlay):

* Вид: Всплывающая плашка со скругленными углами цвета \#1A1A1A, перекрывающая часть интерфейса при вызове кнопками «Вверх/Вниз».  
* Пункты (Навигация кнопками, изменение значений энкодером):  
  1. Целевая температура (Уставка).  
  2. Коррекция топлива (Trim ±%).  
  3. Переключатель АВТО / РУ.  
  4. Переход в Главное Меню.

4\. Органы управления для понимания UX

*Управление устройством производится исключительно блоком физических кнопок(5шт \-* ON/OFF, Вверх, Вниз, Насос, Свеча*) и энкодером(вращение+клик).*

* Энкодер: Вращение без меню — переключение ступеней 1-2-3-4. Вращение в меню — изменение параметров. Клик — возврат в АВТО (если были в РУ) или подтверждение. Длинный клик — вход в меню настроек.  
* Кнопки Вверх/Вниз: Вызов «Быстрого меню» и перемещение по нему.  
* Кнопки Насос/Свеча: Мгновенный переход в РУ с прямым включением/выключением исполнительного механизма (приоритет оператора).  
* Кнопка ON/OFF: Запуск/Остановка по алгоритму. Удержание — аварийный безусловный стоп.

5\. Логика

Логика управления вне меню

* Энкодер (вращение): Переключение фиксированных ступеней мощности от 1 до 4 (в ручном режиме).  
* Энкодер (клик): Быстрый возврат в автоматический режим из ручного.  
* Кнопки Вверх и Вниз: Вызов быстрого меню и перемещение по нему.  
* Кнопки Насос и Свеча: Прямое включение и выключение исполнительных механизмов. Мгновенно переводят систему в ручной режим.  
* Кнопка ON/OFF: Запуск и остановка по алгоритму. Длительное нажатие — аварийный останов.

---

Логика работы Быстрого меню

* Вид: Всплывающий блок со скругленными углами поверх основного интерфейса.  
* Навигация: Кнопки Вверх и Вниз перемещают курсор по пунктам.  
* Изменение значений: Вращение энкодера меняет параметр выбранной строки.  
* Выход: Клик энкодера или автоматическое закрытие через 5 секунд бездействия.  
* Пункты меню:  
  1. Целевая температура (выбрана по умолчанию при открытии кнопкой Вверх).  
  2. Коррекция подачи топлива (в процентах от базовой).  
  3. Переключатель режимов Авто и Ручной.  
  4. Переход в Главное меню (выбран по умолчанию при открытии кнопкой Вниз).

---

Логика работы Главного меню

* Вход: Осуществляется длительным нажатием на энкодер в течение 2 секунд.  
* Назначение: Глубокие настройки системы (калибровка ШИМ для ступеней, лимиты времени свечи и т.д.).
