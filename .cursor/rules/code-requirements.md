# Стандарт оформления C++ кода

При генерации или рефакторинге кода строго следуй данным правилам:

## 1. Именование членов класса
- Все члены классов (private/protected) должны иметь префикс `m_`.
- Формат: `Type m_name;`
- **Пример:** `float m_health;`

## 2. Структура классов
- Всегда отделяй переменные-члены от методов (функций) разными секциями доступа. 
- Даже если в классе уже есть секция `private:`, создай отдельную секцию `private:` специально для данных в конце или в начале блока.
- **Пример:**
  ```cpp
  private:
      void internalHelper();
  
  private:
      int m_speed;
  ```

## 3. Расстановка фигурных скобок
- Открывающая фигурная скобка `{` должна находиться **на той же строке**, что и объявление условия, цикла или функции.
- **Пример:** `if(condition){` или `void solve(){`.

## 4. Пробелы в операторах управления
- **Запрещено** ставить пробел между ключевым словом (`if`, `for`, `while`, `switch`) и открывающей круглой скобкой.
- Правильно: `if(a > b){`
- Неправильно: `if (a > b) {`

### Примеры кода для соблюдения стиля:

**Правильно:**
```cpp
class Player{
public:
    void update(){
        if(m_isActive){
            process();
        }
    }

private:
    void process();

private:
    bool m_isActive;
};
```
